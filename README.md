# Science Olympiad Jr. Explorer

**An interactive learning platform that makes elementary Science Olympiad fun, accessible, and engaging through AI-powered personalized learning.**

---

## Our Mission

Science Olympiad Jr. Explorer was created to solve one of the biggest challenges faced by elementary Science Olympiad teams: the lack of engaging, high-quality practice resources.

Our goal is to help students build confidence, truly understand scientific concepts, and enjoy the learning process—not simply memorize answers. By combining adaptive practice, AI tutoring, visual explanations, and interactive learning, we're making Science Olympiad easier to start and more exciting to continue.

---

## Project Status

**Currently under active development**

The first MVP is being built for pilot testing with Science Gurus elementary Science Olympiad teams. Student feedback will directly shape future features and improvements.
